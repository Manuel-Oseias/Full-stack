Cabeçalhos no HTML

HTML títulos são títulos ou legendas que você deseja exibir em uma página da web.

Cabeçalhos HTML são definidos com o <h1> de <h6> tags. <h1> define o mais importante título. <h6> define o menos importante título.

Nota: os Navegadores automaticamente adicionar algum espaço em branco (a margem) antes e depois de um título.

Títulos São Importantes

Os motores de busca utilizam os títulos para indexar a estrutura e o conteúdo de suas páginas da web.

Muitas vezes, os usuários desnatado uma página por seus títulos. É importante o uso de títulos para mostrar a estrutura do documento.

<h1> os títulos devem ser usados para títulos principais, seguidos por <h2> títulos, então o menos importante <h3>, e assim por diante.

Nota: Usa headings HTML para apenas cabeçalhos. Não utilizar cabeçalhos para tornar o texto GRANDE ou negrito.
