  1. Background-image de forma dêgrade

Para criar dêgrade precisa-se de 2 ou mais core, logo, o background de forma dêgrade é mais nada do que a junção de mais cores. Para criar dêgrade, usamos 2 estilos:

  • Estilo internal
    •
  • Estilo external
    •

  1. Estilo internal

    2. Usa-se background duma única cor no estilo internal da seguinte maneira: 
                                <style>
                                tag(body, div, main, etc..)
                                {
                                    background-image: linear-gradient A POSIÇÃO(to top, to left
                                    , to right, to bottom ), AS CORES (rgb, codigo hexadecimal, nome da cor em ingês);
                                }
                            </style>
                            
    Exemplos de usar Background-color no estilo internal
    
                            Ex01:
                            <style>
                                body
                                {
                                    background-image:linear-gradient(to top, rbg(125,356,234));
                                }
                            <style>

                            Ex02:
                            <style>
                                body
                                {
                                    background-image:linear-gradient(to left, #ff1255);
                                }
                            <style>

                            Ex02:
                            <style>
                                body
                                {
                                    background-image:linear-gradient(to left, blue);
                                }
                            <style>

  2. Estilo external

    3. Usa-se background duma única cor no estilo external da seguinte maneira: 
                                @charset "UFT-8";
                            tag(body, div, main, etc..)
                            {
                                background-image: linear-gradient A POSIÇÃO(to top, to left
                                , to right, to bottom ), AS CORES (rgb, codigo hexadecimal, nome da cor em ingês);
                            }
                            
    Exemplos de usar background-image no estilo external
    
                            Ex01: 
                            body
                            {
                                background-image:linear-gradient(to top, rbg(125,356,234));
                            }
                            Ex02: 
                            body
                            {
                                background-image:linear-gradient(to left, #ff1255);
                            }
                            Ex03: 
                            body
                            {
                                background-image:linear-gradient(to left, blue);
                            }
