  1. Background-image com imagem

Este tema é muito atraente para desenvolvidor dos sites, entrentanto é importante de dizer que A imagem que deve ser colocada no fundo, se for baixada na internet deve ter autirização do autor.. Por padrão qualquer imagens é carregado na sua totalidade ou seja no tamanho pleino. Existe duas maneira de inserir qualquer imagem em qualquer site.

  • Imagem dentro do seu local
    •
  • Imagem na internet
    •

  1. Imagem na internet

Para colocar imagem que está lojado na internet, é necessário que o dono da imagem oferece a permissão e mantenha a imagem no máximo que poder para manter a sua imagem no site, ou seja se aquela imagem na internet foi apagada, automaticamente a sua imagem no site será eliminada.

Para colocar background-image no CSS, usamos 2 estilos princípais:

  • Estilo internal
    •
  • Estilo external
    •

  • Estilo internal

    • Usa-se background-image no estilo internal da seguinte maneira: 

                        <style>
                        tag(body, div, main, etc..)
                        {
                            background-image: url("link da imagem");
                        }
                    </style>
                    
    Exemplos para usar Background-image no estilo internal
    
                    Ex01:
                    <style>
                    tag(body, div, main, etc..)
                    {
                        background-image:linear-gradient(https://www.google.com);            
                    }

  • Estilo external

    • Usa-se background-image no estilo external da seguinte maneira: 

                        tag(body, div, main, etc..)
                    {
                        background-image: url("link da imagem");
                    }
                    
    Exemplos para usar Background-image no estilo internal
    
                    Ex01:
                    tag(body, div, main, etc..)
                    {
                        background-image:linear-gradient(https://www.google.com);            
                    }
