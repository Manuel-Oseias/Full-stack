A palavra background significa fundo em português. O background no HTML não tem conotação semântica, está muito ligado com os estilos ou seja CSS. Existe varias maneiras de usar o background no CSS, neste capitulo veremos as formas mais simples de usar background. os princípais são:

  • Background-color

  1. Background-color

Como o sub-tema é bem especifico que nessa página veremos duas formas princípais de representar os background-color em CSS.

  1. Estilo interno

    2. Usa-se background duma única cor no estilo internal da seguinte maneira: 

                            <style>
                            tag(body, div, main, etc..)
                            {
                                background-color: A COR(rgb, codigo hexadecimal, nome da cor em ingês);
                            }
                        </style>
                        
    Exemplos de usar Background-color no estilo internal
    
                        Ex01:
                        <style>
                            body
                            {
                                background-color: rgb(125,356,234);
                            }
                        </style>
                        Ex02:
                        <style>
                            body
                            {
                                background-color: #ff1255;
                            }
                        </style>
                        Ex03:
                        <style>
                            body
                            {
                                background-color: blue;
                            }
                        </style>

  2. Estilo external

    3. Usa-se background duma única cor no estilo external da seguinte maneira: 

                            @charset "UFT-8";
                        tag(body, div, main, etc..)
                        {
                            background-color: A COR(rgb, codigo hexadecimal, nome da cor em ingês);
                        }
                        
    Exemplos de usar Background-color no estilo external
    
                        Ex01: 
                        body
                        {
                            background-color: rgb(125,356,234);
                        }
                        Ex02: 
                        body
                        {
                            background-color: rgb(125,356,234);
                        }
                        Ex03: 
                        body
                        {
                            background-color: blue;
                        }
