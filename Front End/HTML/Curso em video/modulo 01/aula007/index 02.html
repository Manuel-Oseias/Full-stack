OS LINKS

Links é um navegador gráfico e de texto de software gratuito com um sistema de menu suspenso. Existe 2 tipos de link no html, que são:

  1. Links externos
  2. Links internos

  1. Os links internos

    2. Os links internos são aqueles links que conecta com outros site dentro seu servidor.existe duas maneiras de criar links externos, entretanto os (arquivos)sites devem estar no mesmo servidor ou seja na mesma pasta para ser facilmente localizada. Existe duas maneira de criar links internos:

    1. Os links internos simples

      2. Os links internos simples são aqueles links que na maioria das vezes estão localizada na mesma pasta do (arquivo) site principal, para cria-los é necessário criar um novo (arquivo) site que consequentemente estará alojado dentro da pasta principal. Existe links internos simples fixo e dinámico

      1. Os links internos simples fixo

        2.
        Para criar links internos simples fixo usa-se o comando <a href="Aqui o nome do arquivo(site) target = _self ">A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>

      2. Os links internos simples dinámicos

        3. Para criar links internos simples dinámico usa-se o comando <a href="Aqui o nome do arquivo(site) ref="next/prev/nofollow" target = "_blank" >A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>
