1. Os links internos complexas

    2. Os links internos complexas são aqueles links que na maioria das vezes estão localizada numa sub-pasta do meu site (arquivo) principal, para cria-los é necessário criar uma nova pasta em seguinda um novo (arquivo) site que será alojado dentro da pasta principal. Existe links internos complexas fixo e dinámico

    1. Os links internos complexas fixo

      2. Para criar links internos simples fixo usa-se o comando <a href="Aqui o nome da pasta/arquivo(site)" >A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>

      • Ex: Esta é a terceira página links internos complexa sobre, se quiser aprofundar mais sobre downloard doc via link, basta clicar Aqui

    2. Os links internos complexos dinámico

      3. Para criar links internos complexos dinámico usa-se o comando <a href="Aqui o nome da pasta/arquivo(site)" target = "_blank" rel= "next/prev/nofollow" > A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>
