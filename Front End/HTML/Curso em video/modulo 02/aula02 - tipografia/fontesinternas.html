AS FONTES EM HTML

Existe varias maneiras de colocar fontes em html, algumas desses fontes são:

  1. fonte-family
  2. fonte-size
  3. fonte-style

  1. FONTE-FAMILY

    2. A família de tipos de letra seleção em (X)HTML, CSS, e sistemas derivados especifica uma lista priorizada de fontes e genérico de nomes de família; em conjunto com a correlação de propriedades de tipo de letra, esta lista determina o particular face de tipo de letra utilizado para compor as personagens.

    No CSS há cinco genérico famílias de tipos de letra:

    • Serif font

      • Serif fontes têm um pequeno derrame nas bordas de cada letra. Elas criam uma sensação de formalidade e elegância.
      Ex: Times New roman, Georgia, Garamond, etc...

      Coloca-se fonte serifada (tanto inline,internal ou external) das seguintes maneiras:

      • Inline: <tag(h,p,dt, etc..) style ="font-family: "Nome da fonte serifada", "Nome da fonte serifada", ... serif;">

      • Internal: No corpo do CSS, criar um style{body{"font-family:"Nome da fonte serifada", "Nome da fonte serifada",... serif;"}}

      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-family:"Nome da fonte serifada", "Nome da fonte serifada",... serif;"}



    • Sans Serif font

      • Sans-serif fontes tem linhas limpas (sem pequenos golpes em anexo). Eles criam uma aparência moderna e minimalista.
      Ex: Arial, Verdana, Helvetica, etc...

      Coloca-se fonte não serifada (tanto inline,internal ou external) das seguintes maneiras:

      • Inline: <tag(h,p,dt, etc..) style ="font-family: "Nome da fonte não serifada", "Nome da fonte não serifada", ... serif;">

      • Internal: No corpo do CSS, criar um style{body{"font-family:"Nome da fonte não serifada", "Nome da fonte não serifada",... serif;"}}

      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-family:"Nome da fonte não serifada", "Nome da fonte não serifada",... serif;"}



    • Monospace font

      • Monoespaçada fontes - aqui todas as letras têm a mesma largura fixa. Eles criam uma mecânica olhar.
      Ex: Courier New, Lucida Console, Monaco, etc...

      Coloca-se a fonte monoespaçada (tanto inline,internal ou external) das seguintes maneiras:

      • Inline: <tag(h,p,dt, etc..) style ="font-family: "Nome da fonte monoespaçada", "Nome da fonte monoespaçada", ... serif;">

      • Internal: No corpo do CSS, criar um style{body{"font-family:"Nome da fonte monoespaçada", "Nome da fonte monoespaçada",... serif;"}}

      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-family:"Nome da fonte monoespaçada", "Nome da fonte monoespaçada",... serif;"}



    • Cursive font

      • Cursiva fontes imitar humanos manuscrito.
      Ex: Brush Script MT, Lucida Handwriting, etc...

      Coloca-se fonte cursiva (tanto inline,internal ou external) das seguintes maneiras:

      • Inline: <tag(h,p,dt, etc..) style ="font-family: "Nome da fonte cursiva", "Nome da fonte cursiva", ... serif;">

      • Internal: No corpo do CSS, criar um style{body{"font-family:"Nome da fonte cursiva", "Nome da fonte cursiva",... serif;"}}

      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-family:"Nome da fonte cursiva", "Nome da fonte cursiva",... serif;"}



    • Fantasy

      • Fantasia fontes são decorativos/lúdico fontes.
      Ex: Copperplate, Papyrus


      Nota que qualquer tipo de fonte possui os diferentes nomes de tipo de letra pertence a um dos genéricos famílias de tipos de letra.
      Na mesma linha pode se colocar 3 nomes de fones mas deve sempre terminar com nome generico!Ex: Times New roman, Georgia, Garamond,Sans-serif.

      AS FONTES SERIFADAS SÃO USADA PARA COLOCAR TEMAS E AS FONTES NÃO SERIFADAS SÃO USADAS PARA COLOCAR MATERIAS.

  2. FONT WEIGHT

    3. O peso das fontes são caraterizados por:

    • lighter (entre 100 à 225) - é a fonte mais leve
    • normal (entre 225 à 350) - é a fonte normal
    • bold (entre 300 à 600)- é a fonte escura
    • bolder (entre 600 à 900)- é a fonte mais negritos

    Para a definir o tamanho do texto com pixels oferece controle completo sobre o tamanho do texto, usa-se da seguinte maneira:

    • Inline: <tag(h,p,dt, etc..) style ="font-weight: "Número de tamanho ou nome do peso (lighter, normal, bold, ou bolder)";">

    • Internal: No corpo do CSS, criar um style{body{"font-weight: "Número de tamanho ou nome do peso (lighter, normal, bold, ou bolder)";"}}

    • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-weight: "Número de tamanho ou nome do peso (lighter, normal, bold, ou bolder)";"}

  3. FONT STYLE

    4. A font-style propriedade é principalmente usado para especificar texto em itálico. Esta propriedade tem três valores:

    • Normal - O texto é apresentado normalmente
    • Itálico - O texto é mostrado em itálico
    • Oblíquo - O texto é "encostado" (oblíquo é muito semelhante ao itálico, mas menos suporte)

  4. FONT SIZE

    5. O font-size propriedade define o tamanho do texto. Ser capaz de gerenciar o tamanho do texto é importante no web design. No entanto, você não deve usar o tamanho de fonte ajustes para fazer parágrafos olhar como cabeçalhos ou títulos de olhar como parágrafos. O tamanho do tipo de letra do valor pode ser absoluta ou relativa de tamanho.

    1. Tamanho absoluto

      • Define o texto para um tamanho especificado
      • Não permitir que um usuário altere o tamanho do texto em todos os navegadores (ruim por motivos de acessibilidade)
      • Tamanho absoluto é útil quando o tamanho físico da saída é conhecida


        • Ex: cm, mm, px, in, etc...


    2. Tamanho relativo:

      • Define o tamanho em relação aos elementos circundante
      • Permite que o usuário altere o tamanho do texto em navegadores


        • Ex: em, ex,vw, etc...


      Nota: Se você não especificar um tamanho de tipo de letra, o tamanho padrão para o texto normal, como parágrafos, é 16px (16px=1em).


    Existe 3 maneiras de colocar tamanho no CSS,

    1. Definir O Tamanho Do Tipo De Letra Com Pixels

      2. Para a definir o tamanho do texto com pixels oferece controle completo sobre o tamanho do texto, usa-se da seguinte maneira:

      • Inline: <tag(h,p,dt, etc..) style ="font-size: "Número de tamanho"px;">
      • Internal: No corpo do CSS, criar um style{body{"font-size: "Número de tamanho"px;"}}
      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-size: "Número de tamanho"px;"}

    2. Definir O Tamanho Do Tipo De Letra Com Em

      3. Para a definir o tamanho do texto com Em oferece controle completo sobre o tamanho do texto, usa-se da seguinte maneira:

      • Inline: <tag(h,p,dt, etc..) style ="font-size: "Número de tamanho"em;">
      • Internal: No corpo do CSS, criar um style{body{"font-size: "Número de tamanho"em;"}}
      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-size: "Número de tamanho"em;"}



    3. Ágil Tamanho Da Fonte


      4. O tamanho do texto pode ser definido com uma vw unidade, o que significa que a "janela de visualização de largura".

      De que forma o tamanho do texto seguirão o tamanho da janela do navegador:

      • Inline: <tag(h,p,dt, etc..) style ="font-size: "Número de tamanho"vw;">
      • Internal: No corpo do CSS, criar um style{body{"font-size: "Número de tamanho"vw;"}}
      • External: No corpo do CSS, cria um link doutro estilo e faz a mesma coisa. body{"font-size: "Número de tamanho"vw;"}

JUNTAR TODAS AS FONTES

Existe a possibilidade de juntar 3 ou 4 fontes numa única linha seguindo essa ordem:

  1. font-style
  2. font-weight
  3. font-size
  4. font-family


Usa-se das seguintes maneiras:
  • Inline form

    • < tag(h, p, etc..) style = " font: (font-style fonte) (font-weight) (font-size) ('font-family', 'font-family');" }>

    Ex: <tag(h, p, etc..) style = "font: italic normal 16px 'Arial', 'sens-serif'; }>

    Ex: Lorem ipsum dolor sit amet consectetur adipisicing elit. Eius, tempora, placeat eligendi enim dignissimos voluptatum veritatis accusamus harum maxime laudantium porro fugit! Qui, assumenda quas! Nihil ut nulla voluptatem dolores.

  • Internal form

    • style { tag(h, body, p, etc...) { font: (font-style fonte) (font-weight) (font-size) ('font-family', 'font-family');" } }

    Ex: style { tag(h, p, etc..) { font: italic normal 16px 'Arial', 'sens-serif'; } }

  • External form

    • tag(h,body,p, etc...) { font: (font-style fonte) (font-weight) (font-size) ('font-family', 'font-family');" }

    Ex: tag(h, p, etc..) { font: italic normal 16px 'Arial', 'sens-serif'; }
