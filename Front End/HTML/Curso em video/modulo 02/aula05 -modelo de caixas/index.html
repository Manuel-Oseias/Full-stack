MODELO DE CAIXAS

    Anatomia da caixa

    Em html cada palavra no ecrã se encontra dentro duma caixa, e as caixas são configuradas das seguintes maneiras:

    • Altura
      •  height
      A altura da caixa é representado por palavra height em inglês

    • Largura
      •  width
      A largura da caixa é representado por palavra width em inglês

    • A borda
      •  border
      A borda duma caixa é uma linha que circula o conteúdo, a borda é representado por palavra border em inglês

    • Preenchimento
      •  padding
      Acoxuamento ou preenchimento é um espaço interno que existe entre o conteúdo e a borda ou seja quando a borda fica oferece um espaço do conteúdo, automaticamente cria-se a Acoxuamento. Ele é representado por palavra padding em inglês

    • a margem
      •  margin
      A margim é um espaço exterior que existe entre o conteúdo e a borda. Ele é representado por palavra margin em inglês

    Tipo de caixa

    No HTML, existe 2 tipo das caixas
    • box-level
      •
    • inline-level
      •

      1. Box level

        2. Aquela caixa que sempre se inicia com uma nova linha e também ocupa a largura total da tela.
        Ex: div, (h1...h6), p, main, header, nav, article, aside, footer, form, video, etc...

      2. Inline-level

        3. Aquela caixa que não se inicia com uma nova linha ou seja ele continua na mesma linha, consequentemente não necssariamente ocupa a largura total da tela.
        Ex:span, a, code, small, strong, em, sup, sub, label, button, input, select
