Estilos CSS

A linguagem CSS3 é a linguagem mais especializado no estilo do site, fonte, imagem, etc.. enquanto html está intricicamente ligado com a semântica(texto). Para colocar estilo dentro do CSS3, usa-se 3 maneira básica que são:

  1. Estilos CSS inline
  2. Estilos CSS external
  3. Estilos CSS internal

    Estilos CSS internal

    Quando estuda-se estilos inline, surge a necessidade de melhorar a maneira de criar estilos, porque uma das desvantagem dos estilos inline é a poluição do código html
    Para efectuar estilo interno, não será mais necessário colocar estilo em cada linha do nosso código html, basta configurar no corpo do nosso projeto, tudo será organizado.

    Porém esse método de colocar estilo dentro do corpo faz com que o nosso corpo fica mais grande e longo.
    Para inserir qualquer estilo interno, basta criar um estilo personalizada dentro do nosso html. 

        <style>
                        
            body
            {
                background-color: Nome da cor em ingles.
                font-family: Tipo de letra
                fonte-size: tamanho da letra
            }
            h1
            {
                background-color: Nome da cor em ingles.
                font-family: Tipo de letra
                fonte-size: tamanho da letra
            }
            h1
        </style>
