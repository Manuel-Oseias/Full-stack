Para carrega uma imagem dentro do seu servidor de varios tamanho, precisa do tag:
<picture>
< source media = (max-width:tamanho da foto em px)
 scrset = URLs da imagem ype = image/png>

< source media = (min-width:tamanho da foto em px)
 scrset = URLs da imagem type = image/png>

<image src = URLs de imagem alt = comentário da imagem >
<picture>

Nota que sempre deve começar de ordem crescente ou descencente (min, media, max)

Esse exercício serve para tentar abrir em varios dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.
