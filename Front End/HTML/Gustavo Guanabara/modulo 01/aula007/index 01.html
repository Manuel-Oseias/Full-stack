OS LINKS

Links é um navegador gráfico e de texto de software gratuito com um sistema de menu suspenso. Existe 2 tipos de link no html, que são:

  1. Links externos
  2. Links internos

  1. Os links externos

    2. Os links externos são aqueles links que conecta com outros sites fora do seu servidor.existe duas maneiras de criar links externos.

    1. Os links externos fixo

      2. Os links externos fixo são aqueles que podem ser criada sem ter a necissidade de criar novo abba ou seja ele sobre pôe o site. Para criar qualquer link fixo basta digitar <a href = "Aqui coloca-se o URL do lkink"> a frase que vai ser possivel para usuario clicar <a>

      Ex: Siga o meu canal do youtube clicando aqui

      Ex: Aula sobre manipulação do texto Aqui

      Nota que ao abrir novo link, usuário abriu com a mesma página do mesmo site, isso quer dizer que o desenvolvidor sobre pôs o seu link acima doutro link, na maioria das vezes não é recomendável.


    2. Os links externos dinámico

      3. Os links externos dinámico são aqueles que podem ser criada sem sobre pôr o site ou seja ao ser criada ele abre nova abba. Para criar qualquer link dinámico basta digitar <a href = "Aqui coloca-se o URL do lkink" target = "_blank" rel = "external" > a frase que vai ser possivel para usuario clicar <a>

      Ex: Siga o meu canal do youtube clicando Aqui

      Ex02: Aula sobre manipulação do texto Aqui
