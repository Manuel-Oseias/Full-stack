ALINHAMENTO VERTICAL

Qualquer desevolvidores dos sites simples, sente a necessidade de movementar qualquer caixa, infelizmente o comando margim não nos permite colocar a caixa no centro, e isso dificulta muito. Neste blog, iremos tratar sobre 2 assuntos essencial, alinhamento vertical e altura de viewport, por isso é essencial que o leitor tenha conhecimento básico sobre os background.

Configurar o tamnho do ecrã

Para configurar o tamanho do ercã, usa-se height nos dois estilos das seguintes maneiras: 

                Estilo internal
                ---------------
                <style>
                    tag(body, main, div, etc...)
                    {
                        height: percentagem vh;
                    }    
                </style>
                Exemplo:
                <style>
                    body
                    {
                        height: percentagem vh;
                    }  
                </style>

                Estilo external
                ---------------
                @charset "UTF-8";
                body
                {
                    height: percentagem vh;
                } 
                Exemplo:
                @charset "UTF-8";
                body
                {
                    height: 98vh;
                }

Alinhamento vertical

Quando pretende-se alinhar duas caixas por exemplo, (article e section), se poder obervar na página do exemplo que a primeira caixa está incluido na segunda.usa-se seguintes métodos.

  1. Colocar uma caixa dentro doutra caixa

    2. Para alinhar duas caixa, basta estruturar bem os seus tags dentro do html, por exemplo 

                            <main>
                            <section>
                                <article>
                                    <p>
                                        contéudos
                                    </p>
                                </article>
                            </section>
                        </main>
    Aqui conseguimos notar que a tag, main contain a tag section e a tag section contain a tag article, sucessivamente as tags estão alinhadas dentro duma caixa.

  2. Colocar a caixa maior na configuração relativa

    3. Suponhamos que queremos configurar a tag selection maior e article menor ou seja a tag article está dentro da caixa da tag selection. Essa configuração relativa pode ser feita no estilo internal como external desta maneira: 

                            Estilo internal
                        ---------------
                        <style>
                        selection#maior
                        {
                            position: relative
                        }    
                        </style>

                        Estilo external
                        ---------------
                        @charset "UTF-8";
                        selection
                        {
                            position: relative
                        }

  3. Colocar a caixa maior na configuração relativa

    4. Do mesmo jeito que trabalhamos com exemplo anterior, após de colocar a maior caixa numa posição relativa, aqui colocaremos a menor caixa na posição absoluta afim de conseguir mexer ou movementar a caixa em qualquer posição que queremos.

    Continuamos a trabalhar com exemplo anterior, selection caixa maior e article caixa menor, da mesma forma que colocamos a caixa maior em dois estilo, aqui também faremos o mesmo. 

                            Estilo internal
                        ---------------
                        <style>
                        selection#maior
                        {
                            position: relative DEVEM PERMANECER RELATIVA - para ser fixado
                        }
                        article#maior
                        {
                            position: absolute 
                            left: 50% - posição direta ou esquera 
                            top: 70% - posição a cima ou a baixo
                            transform: translate((70%, 90%) - posição direta ou esquera, posição a cima ou a baixo) 
                        }    
                        </style>

                        Estilo external
                        ---------------
                        @charset "UTF-8";
                        selection#maior
                        {
                            position: relative DEVEM PERMANECER RELATIVA - para ser fixado
                        }
                        selection#menor
                        {
                            position: absolute 
                            left: 50% - posição direta ou esquera 
                            top: 70% - posição a cima ou a baixo
                            transform: translate((70%, 90%) - posição direta ou esquera, posição a cima ou a baixo) 
                        }
