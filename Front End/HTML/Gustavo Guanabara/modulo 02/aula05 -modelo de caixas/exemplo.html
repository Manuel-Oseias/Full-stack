Simplificação e representação

  • Para border
    • invés de escrever:
    • border-width: 10px
    • border-style: solid
    • border-color: #ffffff
    Pode ser simplificada da seguinte maneira:
    boder: tamanho(px, em, vw, etc...) estilo(solid, insert, etc..) cor(branco, beige, etc...)

    Ex: border: 10px insert #ffffff


  • Para padding
    • invés de escrever:
    • padding-top: 10px
    • padding-right: 20px
    • padding-botton: 30px
    • padding-left: 40px
    Pode ser simplificada da seguinte maneira:
    padding: top(px, em, vw, etc...) right(px, em, vw, etc...) botton(px, em, vw, etc...) left(px, em, vw, etc...)

    Ex: padding: 10px 20px 30px 40px
    Ex: padding: 10px
    Ex: padding: 10px (top, botton) 20px (right, left)

  • Para margin
    • invés de escrever:
    • margin-top: 10px
    • margin-right: 10px
    • margin-botton: 10px
    • margin-left: 10px
    Pode ser simplificada da seguinte maneira:
    margin: top(px, em, vw, etc...) right(px, em, vw, etc...) botton(px, em, vw, etc...) left(px, em, vw, etc...)

    Ex: margin: 10px 10px 10px 10px
    Ex: margin: 10px auto(para centralizar) 10px auto (para centralizar)

    Para transformar qualquer box-line em inline-box usa-se o comando: 

                    tag(h,main, div, etc..) 
                { 
                    display: inline
                    
                }
