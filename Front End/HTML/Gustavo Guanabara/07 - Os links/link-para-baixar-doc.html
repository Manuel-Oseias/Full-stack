Baixar os documentos com link

                Anterior

  1. Leitura simple

    2. A leitura simple é aquela leitura que desenvolvidor fez com que o usuário ao clicar no link não possa abrir nova abba, ou seja sobre por o site um documento.

    Para fazer leitura simple online basta digitar <a href="nome do arquivo(livro, pdf, word)" > A mensagem para ser clicado <a>

    • Ex: Clique aqui para baixar livro de Genesis 3

  2. Leitura complexa

    3. A leitura complexa é aquela leitura que desenvolvidor fez com que o usuário ao clicar no link possa abrir nova abba.

    Para fazer uma leitura online duma maneira complexa, basta dar este comando <a href = "nome do arquivo(pdf, word,etc...)" target ="_blank" rel ="external, nofollow, etc...">A frase que quer envelopar ou seja que vai te servirá usuário para clicar<a>

    • Ex01: Clique aqui para abrir nova abba de leitura do Genesis 3.
      •

  3. Para solicitar o downloar automática

    4. Para solicitar downloar duma maneira automática, basta invocar <a href = "nome do arquivo(pdf, word,etc...)" downloar ="nome do arquivo(pdf, word,etc...)" type ="tipo de documento (aplication/pdf)" ">A frase que quer envelopar ou seja que vai te servirá usuário para clicar<a>
