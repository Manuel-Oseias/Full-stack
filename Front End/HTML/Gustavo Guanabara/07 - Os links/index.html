  1. Os links internos

Os links internos são aqueles links que conecta com outros site dentro seu servidor.existe duas maneiras de criar links externos, entretanto os (arquivos)sites devem estar no mesmo servidor ou seja na mesma pasta para ser facilmente localizada. Existe duas maneira de criar links internos:

  1. Os links internos simples

Os links internos simples são aqueles links que na maioria das vezes estão localizada na mesma pasta do (arquivo) site principal, para cria-los é necessário criar um novo (arquivo) site que consequentemente estará alojado dentro da pasta principal. Existe links internos simples fixo e dinámico

  • Os links internos simples fixoO

Para criar links internos simples fixo usa-se o comando <a href="Aqui o nome do arquivo(site) target = _self ">A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>

  1. Os links internos simples dinámicos

Para criar links internos simples dinámico usa-se o comando <a href="Aqui o nome do arquivo(site) ref="next/prev/nofollow" target = "_blank" >A frase que quer envelopar ou seja que vai te servirá usuário para clicar</a>
