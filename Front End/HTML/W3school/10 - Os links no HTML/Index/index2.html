O Atributo dos Links em HTML -

Por padrão, a página vinculada será exibida na janela do navegador atual. Para alterar isso, você deve especificar outro destino para o link.

  1. O atributo target

    2. O target atributo especifica onde abrir o documento vinculado. O target atributo pode ter um dos seguintes valores:

    • _self - Padrão. Abre o documento na mesma janela/aba de como ele foi clicado
    • _blank - Abre o documento em uma nova janela ou guia
    • _parent - Abre o documento no quadro de pai
    • _top - Abre o documento no corpo inteiro da janela

  1. Atributo dowload


O download atributo especifica que o destino (o arquivo especificado no href atributo) serão baixadas quando um usuário clica no hiperlink.
Opcionais valor do download atributo será o novo nome do arquivo após o download. Não há restrições sobre os valores permitidos, e o navegador irá automaticamente detectar a extensão de arquivo correta e adicioná-lo ao arquivo (.img, .pdf, .txt .html, etc.).
Se o valor for omitido, o nome do arquivo original é utilizado.

<download="nome do arquivo">

  1. hAttributo da língua reflang


O hreflang atributo especifica o idioma do documento vinculado. Este atributo é usado somente se o href atributo é definido.
Nota: Este atributo é meramente consultivo.


Para ler todos os códigos de idiomas disponíveis, consulte o nosso código de Idioma de referência.


  1. Atributo da politica referência


O referrerpolicy atributo especifica quais informações de referência para enviar quando o utilizador clica na hiperligação.

Valores De Atributo
Valor Descrição
no-referrer Não há informações de referência é enviado
no-referrer-when-downgrade Padrão. Envia a origem, o caminho e a seqüência de caracteres de consulta se o protocolo de segurança de nível permanece o mesmo ou é mais elevada (HTTP para HTTP, HTTPS HTTPS, HTTP para HTTPS estiver ok). Envia nada para o menor nível seguro (HTTPS não está ok)
origin Envia a origem (esquema, host e porta) do documento
origin-when-cross-origin Envia a origem do documento para solicitação de origem cruzada. Envia a origem, o caminho e a seqüência de caracteres de consulta para a mesma origem pedido
same-origin Envia uma referência para a mesma origem do pedido. Envia nenhuma referência para solicitação de origem cruzada
strict-origin-when-cross-origin Envia a origem se o protocolo de segurança de nível permanece o mesmo ou é mais elevada (HTTP para HTTP, HTTPS para HTTPS e HTTP para HTTPS estiver ok). Envia nada para o menor nível seguro (HTTPS)
unsafe-url Envia a origem, o caminho e a seqüência de caracteres de consulta (independentemente de segurança). Utilize este valor com cuidado!

  1. Atributo rel

O rel atributo especifica o relacionamento entre o documento atual e o documento vinculado.
Usado somente se o href atributo estiver presente.
Dica: mecanismos de Pesquisa pode utilizar este atributo para obter mais informações sobre um link!

Valores De Atributo
Valor Descrição
alternate Fornece um link para uma representação alternativa do documento (por exemplo, imprimir a página, traduzido ou espelho)
author Fornece um link para o autor do documento
bookmark Permanente URL usada para bookmarking
external Indica que o referido documento não é parte do mesmo site como o documento actual
help Fornece um link para um documento de ajuda
license Fornece um link para informações de licenciamento para o documento
next Fornece um link para o próximo documento da série
nofollow Links para uma unendorsed documento, como um link pago. ("nofollow" é usado por Google, para especificar que a pesquisa do Google aranha não deve seguir esse link)
noopener Exige que qualquer navegação contexto criado por seguir o hiperlink não deve ter um abridor de navegação contexto
noreferrer Faz o referente desconhecido. Não cabeçalho referer serão incluídos quando o usuário clica no hiperlink
O documento anterior em uma seleção
search Links para uma ferramenta de pesquisa para o documento
tag Uma tag (palavra-chave) para o documento atual
