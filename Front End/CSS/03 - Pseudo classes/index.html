Pseudo-classes em CSS

Existe duas maneiras de usar pseudo dentro do CSS.

  • Pseudo normal
  • Pseudo Child

  1. PSEUDO NORMAL

    2. Existe um comando chamado de houver pode ser usado para modificar o estilo de qualquer tag quando passar a mouse em cima.
    Pode se criar a forma hover de varias maneiras.

    • Forma internal
      •
    • Forma external
      •

    1. Forma internal

      2. Cria-se qualquer pseudo dentro do CSS na forma interna da seguinte maneira: 

                              <style>
                        {
                            tag(h,p,div, body,span etc...): hover
                            {
                                background-color: nome do fundo em ingles ou código hexadecimal
                                color:nome da cor em ingles ou código hexadecimal
                            }
                        }

      Exemplo usando hover
      Quando passará a mousse aqui em cima, a cor da letra mudará 😍, EXPERIMENTA!!!


    2. Forma external

      3. Cria-se qualquer pseudo dentro do CSS na forma externa da seguinte maneira: 

                              @charset "UTF-8"
                        body(h,p,div, span etc...): hover
                        {
                            background-color: nome da cor em ingles ou código hexadecimal
                        }

  2. PSEUDO CHILD

    3. A forma child é traduzido como a forma criança, serve para esconder um texto ou estilo por debaixo de outro texto. Pode se criar a forma child de varias maneiras.

    • Forma internal
      •
    • Forma external
      •

    1. Forma internal

      2. Cria-se qualquer pseudo child dentro do CSS na forma interna da seguinte maneira: 

                              <style>
                        {
                            tags(h,p,div,body,span etc...): >outros tags que ficará escondido(h,p, div, span, etc...)
                            {
                                display:none; --< Isto para esconder a mensagem.
                            }


                            tags(h,p,div,body,span etc...): >outros tags que ficará escondido(h,p, div, span, etc...)
                            {
                                display:block; --< Isto para apresentar a mensagem.

                                background-color: nome do fundo em ingles ou código hexadecimal
                                color:nome da cor em ingles ou código hexadecimal
                            }
                        }
      Exemplo usando child
      Quando passará a mousse em cima do texto, UMA MENSAGEM ESCONDIDA APARECERÁ

      VOCÊ É DUVIDOSO! VIU E ACREDITOU?


    2. Forma external

      3. Cria-se qualquer pseudo child dentro do CSS na forma external da seguinte maneira: 

                              @charset "UTF-8";

                        tags(h,p,div,body,span etc...): >outros tags que ficará escondido(h,p, div, span, etc...)

                            {
                                display:none; --< Isto para esconder a mensagem.
                            }


                            tags(h,p,div,body,span etc...): >outros tags que ficará escondido(h,p, div, span, etc...)

                            {
                                display:block; --< Isto para apresentar a mensagem.

                                background-color: nome do fundo em ingles ou código hexadecimal
                                color:nome da cor em ingles ou código hexadecimal
                            }
