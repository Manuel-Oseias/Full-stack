  2. Alinhamento interno

    • Alinhamento interno à esquerda
      • Para fazer um alinhamento inline à esquerda, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: justify; }
    • Alinhamento interno à direta
      • Para fazer um alinhamento inline à direta, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: direta; }
    • Alinhamento interno no centro
      • Para fazer um alinhamento inline à no centro, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: centrer; }
    • Alinhamento interno ampliada
      • Para fazer um alinhamento inline ampliada, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-indent: número de espaço em px; }

  3. Alinhamento externo

    • Alinhamento externo à esquerda
      • Para fazer um alinhamento externo à esquerda, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: justify; }
    • Alinhamento externo à direta
      • Para fazer um alinhamento externo à direta, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: direta; }
    • Alinhamento externo no centro
      • Para fazer um alinhamento externo à no centro, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text-align: centrer; }
    • Alinhamento externo ampliada
      • Para fazer um alinhamento externo ampliada, usa-se os seguintes códigos:

      style { tag(h,p,dt,etc...) text- indent: número de espaço em px; }
