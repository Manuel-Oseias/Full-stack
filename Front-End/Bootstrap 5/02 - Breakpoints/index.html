Breakpoints para Criação de Leiautes Responsivos

Qualquer site precis dum design responsivo que o permitirá de apresentar em qualquer tela o conteúdo da imagem. Não podemos tratar da responsividade no Bootstraps sem falar primordialmente dos Breakponts

Os breakpoint representam algumas possivéis larguras da tela, o sistema layout dos breakpoint no Bootstrap tem uma caracteristica chamada de mobile first ou seja ele prioriza os aparelho com layout pequenos e o desenvolvidor deve acrescentar se quiser o layout de grande porte.

Veja a table caracteristica do breakpoint

Breakpoints
Breakpoint Infixo de classe Dimensões
Extra-pequeno Nenhum < 576px
Pequeno sm ≥ 576px
Médio md ≥ 768px
Grande lg ≥ 992px
Extra-grande xl ≥ 1200px
Extra-extra-grande xxl ≥ 1400px

Para configurar o nosso site usando essa tabela, basta criar uma class e adicionar o seguintes parametro

atributo- infixo de classe (pode ser sm ou md etc..) - atributo
