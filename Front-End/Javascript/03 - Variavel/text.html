Os Operadores

Oprerador de atribuição

Em JavaScript, o sinal de igual ( = ) é uma atribuição do operador, e não uma igual a do operador. Isso é diferente de álgebra.

Ex: nome = 'Manuel'; - Aqui estamos atribuir valor Manuel na variável nome
Ex: idade = 90; - Aqui estamos atribuir o valor 90 na variável idade

Qualquer valores atribuido pode ser mútavel ou substituido

Para representar sinal de igualidade usa-se ( == ) igual duas vezes.

Ex: Manuel == 'Manuel'; - Aqui estamos dizer que Manuel é igual Manuel
Ex: 90 == 90; - Aqui estamos dizer que 90 é igual 90

Qualquer valores atribuido NÃO pode ser substituido e DEVE TER MESMO CERTAS CARACTERISTICAS

Operador de adição ou a concatenação

Em JavaScript, o sinal de adição + pode ser uma concatenação do operador, com seguintes condições

  • Quando está no meio de duas variáveis do tipo string
    • Ex:
    nome = Manuel nome2 = Oséias
  • Quando está no meio de duas variáveis armazenado involuntáriamente como string
